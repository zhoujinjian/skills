# 测试场景遗漏检查清单

本文档包含11个维度的详细检查项，用于逐维度扫描已有测试用例的场景覆盖情况。

---

## 维度1：等价类覆盖

### 有效等价类检查
- [ ] 每个输入字段的有效等价类是否都有对应用例
- [ ] 多个有效等价类的组合场景是否覆盖
- [ ] 不同用户角色/权限下的有效操作是否覆盖

### 无效等价类检查（最易遗漏）
- [ ] 每个输入字段的无效等价类是否完整（类型错误、格式错误、范围越界、空值）
- [ ] 多个字段同时为无效值的组合场景
- [ ] 有效值与无效值混合输入的场景
- [ ] 不支持的数据类型输入（如数值字段输入文字、日期字段输入非法日期）
- [ ] 超长输入、超短输入、空输入是否分别有独立用例

### 常见遗漏场景示例
- 手机号输入：缺少+86前缀、含空格、含字母、12位号码、1开头的非有效号段
- 邮箱输入：缺少@符号、多个@符号、@后无域名、特殊字符、超长邮箱
- 金额输入：负数、零、超精度小数、千分位格式、科学计数法
- 枚举字段：传不在枚举范围内的值、传空字符串、传数字代替字符串

---

## 维度2：边界场景

### 边界值检查
- [ ] 每个有范围限制的字段是否覆盖：最小值-1、最小值、最小值+1、最大值-1、最大值、最大值+1
- [ ] 字符串长度边界：0字符、1字符、最大长度-1、最大长度、最大长度+1
- [ ] 数值边界：0、-1、最小负数、最大正数、溢出值
- [ ] 日期时间边界：最早日期、最晚日期、2月29日（闰年）、跨年、跨时区

### 空值/特殊值检查（最易遗漏）
- [ ] 所有必填字段为空是否覆盖
- [ ] 所有选填字段为空是否覆盖
- [ ] null/undefined/None值是否覆盖
- [ ] 全空格、全制表符、纯换行符
- [ ] 零宽度字符、Unicode控制字符
- [ ] 特殊字符：`< > & " ' / \ | : * ? ! @ # $ % ^ ( ) [ ] { } ; , . ~ \` + = -_`

### 常见遗漏场景示例
- 密码：1位密码、2位密码、最大长度密码、最大长度+1密码、全空格密码
- 手机号：10位、11位、12位、含特殊字符、纯数字但非手机号格式
- 文件上传：0字节文件、恰好限制大小的文件、超限1字节的文件
- 分页：第0页、第1页、最后一页、超出总页数的页码

---

## 维度3：业务规则交叉

### 条件组合检查
- [ ] 所有业务条件的两两组合是否覆盖（成对测试法）
- [ ] 所有条件同时满足的场景
- [ ] 所有条件同时不满足的场景
- [ ] 部分条件满足部分不满足的排列组合
- [ ] 条件优先级冲突时的处理逻辑

### 业务流程交叉检查
- [ ] 正常流程与异常流程的交叉点
- [ ] 并发操作同一资源的场景
- [ ] 操作过程中状态变更的场景（如编辑时被他人删除）
- [ ] 多步骤流程中各步骤的异常回退

### 常见遗漏场景示例
- 优惠券叠加：满减券+折扣券+免邮券是否可同时使用
- 权限交叉：有查看权限但无编辑权限时的操作行为
- 订单取消：已支付待发货 vs 已发货待收货的取消逻辑差异
- 多角色操作：管理员删除资源时普通用户正在编辑该资源

---

## 维度4：状态迁移

### 状态转换检查
- [ ] 所有合法状态转换路径是否覆盖
- [ ] 所有非法状态转换是否有拦截用例
- [ ] 状态回退场景是否覆盖（如：已审核→待审核）
- [ ] 状态跳转场景是否覆盖（如：待审核→已完结，跳过审核中）
- [ ] 并发状态变更的竞态条件

### 异常状态检查（最易遗漏）
- [ ] 中间状态超时后的处理
- [ ] 外部系统调用失败后的状态回退
- [ ] 网络中断后的状态恢复
- [ ] 重复触发同一状态变更（如重复提交、重复支付）
- [ ] 状态数据不一致的检测和修复

### 常见遗漏场景示例
- 订单状态：待支付→已取消后再次支付、已发货→退款中的转换
- 审批流：审批中发起人撤回、审批通过后上级驳回
- 支付状态：支付中网络超时→状态未知→手动查询确认
- 账号状态：正常→冻结→解冻→冻结，循环状态的处理

## 维度5：非功能性需求

### 性能场景
- [ ] 高并发操作是否有对应用例（如100人同时登录、1000人同时下单）
- [ ] 接口响应时间是否有验证用例
- [ ] 大数据量操作是否有对应用例（如10万条记录的列表查询）
- [ ] 长时间运行的稳定性（如7×24小时连续运行）
- [ ] 资源耗尽场景（如磁盘满、内存不足、连接池耗尽）

### 安全场景
- [ ] SQL注入：输入框输入 `' OR 1=1 --` 等注入语句
- [ ] XSS攻击：输入框输入 `<script>alert('XSS')</script>` 等脚本
- [ ] CSRF攻击：跨站请求伪造的防护验证
- [ ] 越权访问：水平越权（访问他人数据）、垂直越权（低权限访问高权限功能）
- [ ] 密码安全：密码是否明文传输、是否可暴力破解、密码复杂度规则
- [ ] 敏感信息：日志中是否打印敏感信息、接口返回是否泄露敏感字段
- [ ] 文件上传漏洞：上传可执行文件、上传超大纲马、文件名注入

### 兼容性场景
- [ ] 浏览器兼容：Chrome、Firefox、Safari、Edge、IE（如需支持）
- [ ] 设备兼容：iOS、Android、PC、平板
- [ ] 分辨率兼容：不同屏幕尺寸和分辨率
- [ ] 网络兼容：WiFi、4G、5G、弱网、断网
- [ ] 操作系统兼容：Windows、macOS、Linux、不同系统版本

---

## 维度6：性能场景（详细）

### 接口性能
- [ ] 单接口响应时间：正常负载下是否满足SLA
- [ ] 接口并发能力：逐渐加压至目标QPS，验证响应时间和成功率
- [ ] 接口混合场景：多接口按业务比例并发调用

### 数据量性能
- [ ] 大数据量列表查询（分页加载、筛选排序）
- [ ] 大数据量导出（超10万条数据导出）
- [ ] 大数据量导入（批量导入的耗时和成功率）

### 资源性能
- [ ] 连接池耗尽场景
- [ ] 内存泄漏场景（长时间运行后内存是否持续增长）
- [ ] 文件句柄泄漏场景

---

## 维度7：安全场景（详细）

### 注入攻击
- [ ] SQL注入：登录框、搜索框、URL参数、HTTP头
- [ ] 命令注入：系统命令执行点
- [ ] LDAP注入：目录服务查询点
- [ ] XML注入：XML解析入口

### 跨站攻击
- [ ] 反射型XSS：URL参数回显场景
- [ ] 存储型XSS：用户输入存储后展示场景
- [ ] DOM型XSS：前端JS动态渲染场景
- [ ] CSRF：敏感操作的Token验证

### 认证与授权
- [ ] 未登录访问需认证接口
- [ ] Token过期后的处理
- [ ] Token伪造/篡改
- [ ] 水平越权：修改ID访问他人数据
- [ ] 垂直越权：普通用户访问管理员接口
- [ ] 密码暴力破解防护
- [ ] 验证码绕过

### 数据安全
- [ ] 密码是否明文传输和存储
- [ ] 敏感字段是否脱敏展示
- [ ] 接口返回是否包含多余敏感字段
- [ ] 日志是否记录敏感信息
- [ ] 文件上传类型和大小限制

---

## 维度8：反向用例

### 异常分支检查
- [ ] 每个if-else分支是否都有对应用例
- [ ] 每个catch/exception是否都有独立触发用例
- [ ] 每个错误码是否有独立验证用例
- [ ] 第三方服务调用失败的降级处理
- [ ] 数据库操作失败的回滚处理

### 常见遗漏的反向用例
- [ ] 重复操作（重复提交、重复删除、重复支付）
- [ ] 超时操作（请求超时、支付超时、会话超时）
- [ ] 依赖服务不可用（缓存服务宕机、消息队列不可用）
- [ ] 数据异常（脏数据、数据格式不符、关联数据缺失）
- [ ] 幂等性验证（同一请求重复调用结果是否一致）

---

### 浏览器兼容
- [ ] Chrome最新版 + 前两个大版本
- [ ] Firefox最新版 + 前两个大版本
- [ ] Safari最新版（macOS/iOS）
- [ ] Edge最新版
- [ ] 微信内置浏览器（如需支持）
- [ ] IE11（如需支持）

### 设备兼容
- [ ] iPhone各尺寸（SE/标准/Plus/Pro Max）
- [ ] Android各尺寸（小屏/标准/大屏/折叠屏）
- [ ] iPad各尺寸
- [ ] PC各分辨率（1366×768/1920×1080/2K/4K）

### 网络兼容
- [ ] 弱网环境（3G/2G模拟）
- [ ] 网络切换（WiFi↔4G↔5G）
- [ ] 网络中断后恢复
- [ ] 高延迟环境（>3s）
- [ ] 请求超时后的重试机制

## 维度10：历史BUG案例

### BUG复盘检查（需要用户提供历史BUG数据）
- [ ] 每个历史BUG是否有对应的回归用例
- [ ] 同类BUG是否在其他模块也存在
- [ ] 根因分析中的修复点是否有直接验证用例
- [ ] BUG触发条件的变化组合是否覆盖
- [ ] 修复后是否引入新问题的验证

### 常见高频BUG模式
- 空指针异常：对象为空时直接调用方法
- 数组越界：边界值未正确处理
- 并发问题：多线程/多进程操作共享资源
- 缓存问题：缓存与数据库数据不一致
- 时区问题：不同时区时间计算错误
- 编码问题：中文/特殊字符编码乱码
- 精度丢失：浮点数计算精度问题

---

## 维度11：业务特定规则

### 隐藏规则检查（需要用户提供业务规则文档）
- [ ] 文档中明确列出的规则是否都有对应用例
- [ ] 规则之间的冲突场景是否覆盖
- [ ] 规则的例外情况是否覆盖
- [ ] 规则触发条件的边界是否覆盖

### 禁止规则检查
- [ ] 每条禁止规则是否有反向验证用例
- [ ] 尝试绕过禁止规则的场景是否覆盖
- [ ] 禁止规则的提示信息是否正确
- [ ] 禁止规则被违反后的系统状态是否正确

### 常见业务隐藏规则
- 金额上限限制（单笔/日累计/月累计）
- 操作频率限制（每分钟/每小时/每天最大次数）
- 用户等级限制（不同等级不同权限/额度）
- 时间窗口限制（仅在工作时间/指定时段可操作）
- 地域限制（特定地区不可用/特殊处理）
